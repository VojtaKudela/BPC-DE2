Detailed Description

The function createGUImain creates a GUI for connecting, setting the time, and displaying graphical results.

This function GUI_create creates the application's GUI. It calls the functions createGUImain, createGUIset, and createGUIshow, which create the individual sections of the window.

The function createGUIshow creates a GUI for visual display of measured values of temperature, humidity, soil moisture and lighting.

The function createGUIset creates a GUI for regulation of temperature, humidity, soil moisure and illumination.

import createGUImain

The function creates three buttons in the upper left corner of the self: Connect, Set time, and Graphical results.

import createGUIset

A panel will be created in the upper right corner of the self. The panel will contain four buttons: Set temperature, Set humidity, Set soil moisture and Set illumination. The panel will also display the set values including hysteresis.

import createGUIshow

The function creates a panel at the bottom of the window. The window will contain text fields that display the measured value of all four monitored values. It also displays the values using graphic indicators.
