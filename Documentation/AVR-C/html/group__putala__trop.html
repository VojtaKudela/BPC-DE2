◆ read_dht_data()

void read_dht_data ( void )

Reads data from the combined humidity and temperature sensor.

Parameters
None
Returns
None

Communication is via I2C. In the first step, a request is sent to write to the real-time clock with the memory address. Then a request is sent to read. The contents of the first 7 memories are read: seconds, minutes, hours, days of the week, date, month and year.

void read_rtc_data ( void )

Reads data from the real-time clock.

Parameters
None
Returns
None
Note
If hum_int + temp_int + (hum_dec + temp_dec)/10 is not cumsum, data are not overwriten.

Communication is via I2C. In the first step, a request is sent to write to the combined humidity and temperature sensor with the memory address. Then a request is sent to read. The contents of all 5 memories are read: hum_int, hum_dec, temp_int, temp_dec, cumsum.

void startup ( void )

Initialization of peripherals.

Parameters
None
Returns
None

Sets the last character of the transmitted strings. Sets the default values ​​of the control register. Initializes the heater, fan, valve and inhaler. Initializes TWI, UART and ADC. Sets PWM. Sets the interrupts of timers 1 and 2. Enables interrupts.

void write_rtc_data ( void )

Sets the real-time clock value.

Parameters
None
Returns
None

Sends a write request. The first byte is the address of the memory to be overwritten. This is followed by a total of 7 bytes with the values ​​of the memory in which the time is stored.
